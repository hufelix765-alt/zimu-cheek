# 字幕收集

上传字幕、按剧名管理、检索单词的本地网站。

## 功能

- **上传字幕**：填写剧名，上传 SRT / VTT / TXT 等字幕文件
- **随时追加**：同一剧名可多次上传，字幕会累加
- **剧集列表**：查看所有剧名及字幕统计
- **单词检索**：在所有字幕或指定剧集中搜索单词/短语

## 启动

```bash
npm install
npm run dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)

## 数据存储

字幕数据保存在项目目录下的 `data/subtitles.db`（SQLite），无需额外配置数据库。
